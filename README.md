> [!CAUTION]

> WinDivert - это инструмент для перехвата и фильтрации трафика, необходимый для работы zapret.
> Замена iptables и NFQUEUE в Linux, которых нет под Windows.
> Он может использоваться как хорошими, так и плохими программами, но сам по себе не является вирусом.
> Драйвер WinDivert64.sys подписан для возможности загрузки в 64-битное ядро Windows

# ⚙️ ИСПОЛЬЗОВАНИЕ
1. Включете Secure DNS
    - В Chrome - "Использовать безопасный DNS", и выбрать поставщика услуг DNS
    - В Firefox - "Включить DNS через HTTPS, используя: Максимальную защиту", затем "Выбрать поставщика" и вписать URL поставщика
    - В Windows 11 поддерживается включение Secure DNS прямо в настройках ОС
    
2. Скачайте архив со [страницы последнего релиза](https://github.com/FunsyMe/Orbitus-Service/releases/latest)
3. Распакуйте содержимое архива по пути, который не содержит кириллицу или другие спец. символы
4. Запустите нужную конфигурацию zapret из папки `pre-configs`

# ℹ️ ОПИСАНИЕ ФАЙЛОВ
- `general*.bat` - запуск конфигурации zapret вручную
- `orbitus_service.bat` - утилита для управления стратегиями zapret
  - `Установить сервис` - установка и запуск любой конфигурации zapret в автозапуск
  - `Удалить сервис` - удаление и остановка конфигурации zapret из автозапуска

  - `Сменить Game Filter` - переключение режима обхода zapret для игр
    - `TCP, UDP` - под проверку попадают все протоколы 
    - `TCP` - под проверку попадают только TCP протоколы
    - `UDP` - под проверку попадают только UDP протоколы 
  - `Сменить IPset Filter` - переключение режима обхода zapret
    - `none` - никакие IP не попадают под проверку
    - `any` - любой IP попадает под фильтр
    - `loaded` - IP проверяется на вхождение в список
  - `Авто-Проверка обновлений` - переключение автоматического обновления
    - `отключена` - проверка обновлений будет проводиться только вручную
    - `включена` - проверка обновлений будет автаматической
  
  - `Обновить файл IPset` - обновление файла IPset до актуальной версии
  - `Обновить файл hosts` - обновление файла hosts до актуальной версии
  - `Проверить обновления` - обновление и установка утилиты

  - `Диагностика zapret` - диагностика на распространённые причины, по которым zapret может не работать
  - `Авто-Поиск конфигурации` - автоматический поиск конфигураций zapret
